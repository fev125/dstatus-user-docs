# 更新日志

---

## 2026-03-25

### 访问控制

- **`/admin/access-control`**：公开 / 口令 / 严格私有三档，集中管理前台访问与口令、令牌、节点分享。

### 网络质量

- 单节点详情：监控目标列表全量展示，当前时段无数据时灰态占位。

### 管理后台

- 设置中心：基础设置与高级设置切换时不再把页面拉到中间位置；直接访问带 `#tab-*` 的链接也会停留在当前页面，不再首屏原生跳到内容区块。
- 侧边栏：日间下头部蓝底盾牌图标不再被菜单图标色规则覆盖，白标对比恢复。
- 服务器管理：顶级 Tab 条外壳与 `admin-card` + `glass-card` 对齐，与列表卡片/监控页 Tab 壳视觉一致。
- 带图标的 `admin-input--sm` 搜索框：`pl-*`/`pr-*` 与组件内边距层叠修正，SSH 脚本搜索区图标与清除按钮对齐方式优化。
- 分组管理：列表卡片、拖拽手柄、操作按钮与弹窗统一到后台主题 token；暗色和毛玻璃下层次更稳，不再依赖 glass 私有补丁。

## 2026-03-24

### 前台页面
- 详情页、网络质量、IP 质量及部分管理页静态资源显式版本化，便于发布后更新
- 网络质量页 mini 目标列表滚动时去掉底部/顶部渐隐遮罩

### 管理后台
- 高级分析：「AI 设置」与主题 token 对齐；分析 Tab 工具栏与后台中型按钮一致
- GitHub OAuth：授权完成后立即关闭弹窗；默认尺寸略缩小（480×620）

### 服务器管理
- 新增「本地 SSH 保险箱」：密码/私钥/口令在浏览器加密保存，接入测试连接与 SSH 安装/更新；SSH 安装与任务升级通道分离
- 编辑未保存时执行 SSH 操作会先保存；新建 Active 自动安装走 SSH
- 保险箱分区（当前节点 / 本地备份）、导入导出、锁定与清空；支持「迁移到本地」并清理服务器侧认证；口令仅会话保存、不上传服务器的提示
- macOS 用户级 SSH 安装探针改用 launchd 登录自启服务，与卸载口径一致
