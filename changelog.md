## 2026-03-25

### 单节点详情
- 修复口令模式下单节点分享链接打开后被跳回登录的问题，分享页会先完成授权再进入详情
- 修复口令模式下单节点分享页的网络质量、IP 质量等数据读取失败问题，分享页可继续查看当前节点相关数据
- 修复离线节点的单节点分享页顶部概览残留 `NaN` 的问题；当前节点离线时会改为显示离线/未知/0 等可读状态

## 2026-03-24

### 前台页面
- 前台详情页、网络质量页、IP 质量页及部分管理页静态资源改为显式版本化，发布后更容易更新
- 网络质量页 mini 目标列表滚动时不再显示底部/顶部渐隐遮罩层

### 管理后台
- 高级分析页：「AI 设置」与主题 token 对齐；分析 Tab 工具栏按钮字号与内边距收紧，与后台中型按钮一致
- GitHub 登录成功后的弹窗不再倒计时，授权完成后立即关闭；OAuth 弹窗默认尺寸略缩小（480×620）

服务器管理
- 服务器管理页新增「本地 SSH 保险箱」，支持密码、私钥、私钥口令加密保存在浏览器，已接入 测试连接 和 SSH 安装/更新
- SSH 安装/更新 固定走 SSH，升级任务 固定走任务下发，通道不再混用
- 编辑页未保存时点击 SSH 安装/更新 会先保存再执行；新建页选择 Active 后自动安装改为直接走 SSH
- 保险箱按钮按「当前节点 / 本地备份」分区并显示来源，支持导入导出、锁定与清空
- 编辑页支持「迁移到本地」将数据库 SSH 密码/私钥迁到浏览器后再清理服务器认证
- 设置保险箱口令时页面明确提示「口令只保存在当前浏览器会话、不会发送到服务器，忘记后无法找回」
- 修复 macOS 用户级 SSH 安装探针时第 7 步创建服务失败的问题；现在会改用 launchd 创建登录后自启服务，安装与卸载口径保持一致
