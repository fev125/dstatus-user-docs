# 更新日志

## 2026-07-12

### 修复

- 开启域名分离后，管理端为 Agent 生成 SSH 安装/升级与复制安装/重装命令时不再回退到前台域名；默认上报地址改为从当前管理员请求的后台域名解析，避免 Agent 上报和任务轮询被前台域名 404 拦截。

### 优化

- 管理控制台侧边栏去掉 logo 图标，仅保留「管理控制台」标题。
- App 授权二维码改为进入「连接与授权 → App 授权」后自动常驻显示，并按 90 秒轮换；扫码完成后设备授权会保留，仍可在已授权设备列表调整档位或删除设备。
- 控制台「盘满倒计时」改为读取真实硬盘历史，按节点和分区区分预计占满、健康、数据过期与积累中状态。
- 节点详情新增按分区硬盘使用历史，可切换 7 天、30 天、90 天和全部范围；离线或漏采时曲线会保留断点，不再连成虚假平线。

## 2026-07-11

### 新增

- 个性化设置新增「官方主题预设」：一键切换 Vercel 纯黑 / 石墨深灰 / 经典蓝黑 / 暖黑 / 冷黑五套配色，也可自定义强调色与日/夜底色（带明度护栏防误设），随时一键恢复默认。

### 修复

- 网络质量页筛选按钮：日间+毛玻璃模式下选中按钮的高亮底色此前被玻璃效果盖住，现已恢复主题强调色，与夜间行为一致。
- pure PostgreSQL 上线保护增强：PG 连接或数据库迁移失败时不再尝试回落到 SQLite；发布前可用独立备份执行隔离迁移与对账，避免故障写入分叉或缺表仍误判通过。
