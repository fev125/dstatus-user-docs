# 更新日志

---

## 2026-03-23

### 个性化设置

- 主题设置增加「前台自定义 CSS」区块：可启用并将 CSS 持久化到数据库，由服务端向非管理后台页面注入样式（公开同步接口不返回 CSS 正文）
- 用户文档新增专页《主题 CSS 自定义》（侧栏「系统设置」下可直达；线上 <https://docs.vps.mom/theme-custom-css>）

### 管理后台

- 网络监控配置页：监控目标批量操作合并为单条工具栏、顶栏刷新按当前 Tab 生效；目标列表搜索与分组/状态筛选左右排列，统计样式更克制；批量勾选工具区在首次勾选后再显示；监控节点 Tab 搜索在左、分组/状态/排序与整组操作在右
- 网络监控配置页：监控目标列表刷新后会保留当前筛选与页码，保存目标后会自动定位到刚操作的目标；目标弹窗的打开、关闭和预填充逻辑已统一收口，交互更稳定
- 网络监控配置页：监控目标卡片的标题区不再额外渲染内部容器底色与分隔条，卡片背景层次更简洁，后续样式维护也收口到页面级规则
- 网络监控配置页：监控目标弹窗的表单间距已收口到页面级样式 owner，检测类型与分区标题之间的纵向节奏更稳定，后续维护不再依赖模板里多层 `space-y / pt / mb` 叠加
- 网络监控配置页：监控目标卡片已升级为更轻的 A 向信息卡，主机地址支持 code 样式与一键复制，底部操作改为更易扫读的文字按钮
