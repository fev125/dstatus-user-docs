# DStatus 文档中心

欢迎访问 DStatus 分布式监控系统文档中心。这里提供完整的使用指南、功能说明和部署教程。

## 🌟 关于 DStatus

DStatus 是一个企业级分布式服务器监控系统，提供实时性能监控、网络质量分析、智能告警与 AI 驱动的性能优化。

### ✨ 核心优势

#### 🎨 界面优美，沉浸式体验
- **现代化 UI 设计** - 基于 TailwindCSS 构建，渐变色、玻璃拟态、圆角卡片等现代设计元素
- **玻璃拟态效果** - 三种预设档位（轻盈/平衡/丰富），可自定义透明度、模糊强度、饱和度，支持镜头和扭曲特效
- **视频动态背景** - 支持 MP4/WebM/OGG 格式视频背景，打造沉浸式监控体验
- **深色模式** - 完善的亮色/暗色主题切换，护眼舒适
- **响应式布局** - 完美适配桌面、平板、移动设备

#### 🏠 首页功能强大，交互灵活
- **双视图模式** - 卡片模式和列表模式自由切换，满足不同浏览习惯
- **强大筛选** - 支持在线/离线状态、到期时间（3天/7天/30天）、地区、标签等多维度筛选
- **多种排序** - CPU、内存、流量、上传/下载速度、到期时间等指标灵活排序
- **分组筛选** - 按业务、地域等自定义分组快速定位服务器
- **拖拽排序** - 首页直接拖拽调整服务器显示顺序，个性化定制布局
- **隐私模式** - 一键开启隐私模式，保护敏感信息
- **懒加载优化** - 大量节点场景下自动懒加载，保持流畅体验
- **3D 地球可视化** - 炫酷的 3D 地球展示节点全球分布，支持国旗显示和交互

#### 📊 卡片信息丰富，一目了然
- **实时性能** - CPU、内存、磁盘、网络速度等关键指标实时显示
- **网络质量** - 延迟和丢包率显示，可自定义选择监控节点
- **流量统计** - 月流量使用情况、实时上传下载速度
- **到期管理** - 服务器到期时间、剩余价值显示
- **自定义标签** - 支持显示自定义节点标签，灵活分类
- **在线时长** - 服务器运行时间统计

#### 📈 专业图表，灵活多维
- **企业级图表库** - 基于 ECharts 构建，性能卓越，支持海量数据渲染
- **灵活时间范围** - 预设 1小时~30天 等多个时间段，支持自定义时间选择
- **多层级颗粒度** - 分钟级、5分钟级、小时级、天级、月级数据，按需查看
- **专业可视化** - 平滑曲线、区域填充、多 Y 轴、自定义 tooltip、颜色映射稳定
- **实时与历史** - 实时监控与历史分析无缝切换，支持长期趋势观察

#### 🔍 网络质量，精准洞察
- **多节点监控** - 支持从多个探针节点同时监控同一目标
- **延迟与丢包** - TCPing 延迟曲线、丢包率统计、连通性测试
- **范围带显示** - 可视化展示网络波动范围，快速识别不稳定时段

### 核心功能模块

#### 📊 监控与数据采集
- **实时性能监控** - CPU、内存、磁盘、网络、进程等关键指标实时采集与可视化
- **网络质量监控** - TCPing 延迟监控、丢包率分析、多节点连通性测试、范围带可视化
- **流量统计** - 网络流量监控、带宽使用分析、流量趋势图表、多时间段对比

#### 🖥️ 服务器管理
- **服务器管理** - 集中管理所有监控节点，支持批量操作和标签分类
- **分组管理** - 按业务、地域等维度对服务器进行分组，便于统一管理
- **自动发现** - 支持 Agent 自动注册和批量部署，简化大规模部署流程
- **WebSSH 终端** - 浏览器内直接访问服务器终端，无需额外工具
- **SSH 脚本管理** - 创建和管理可复用的 Shell 脚本片段，提升运维效率

#### 🔔 告警与通知
- **多通道告警** - 支持邮件、Telegram、Webhook 等多种通知方式
- **自定义通知任务** - 灵活配置告警规则、阈值和通知频率
- **告警静默** - 支持告警静默和免打扰时段设置

#### 🤖 智能分析
- **AI 性能分析** - Google Gemini 驱动的智能性能分析与优化建议
- **异常检测** - 自动识别性能异常和潜在风险
- **趋势预测** - 基于历史数据预测资源使用趋势

#### 🔐 安全与权限
- **双因素认证** - 支持 TOTP 两步验证，保障账户安全
- **GitHub OAuth** - 支持 GitHub 第三方登录
- **登录通知** - 邮件通知登录行为，及时发现异常
- **HTTP 安全设置** - CSP、HSTS、XSS 防护等安全策略配置

#### ⚙️ 系统配置
- **许可证管理** - 灵活的商业许可证系统，支持多层级授权
- **系统设置** - 站点配置、端口设置、主题切换、Logo 定制
- **高级设置** - 性能优化、PostgreSQL 配置、存储管理、数据清理
- **美化设置** - 个性化主题配置、背景图片、品牌色设置

#### 🗄️ 数据存储
- **双数据库支持** - SQLite（开箱即用）/ PostgreSQL（生产级扩展）
- **数据保留策略** - 灵活配置不同层级数据的保留时长
- **自动归档** - 历史数据自动归档和压缩，优化存储空间
- **数据清理** - 定期清理过期数据，保持系统性能
