# DStatus 页面总览

DStatus 管理后台的功能页面导航。

## 页面导航

### 管理后台主界面
登录后进入主管理界面，左侧边栏显示所有功能模块：

#### 面板管理
- **[服务器管理](/server-management)**：添加、编辑、删除监控服务器
- **[网络监控](/monitor)**：配置监控目标（配置页：面板 `/admin/monitor`；查看页：面板 `/network-quality`）
- **[分组管理](/groups)**：服务器分组和批量管理
- **[自动发现](/autodiscovery)**：自动注册服务器节点
- **[账单报告](/billing-report)**：全局成本概览与到期提醒

#### 系统设置
- **[系统设置](/system-settings)**：基本配置和个性化设置
- **[个性化设置](/personalization)**：控制显示模块、游客可见、壁纸与页脚
- **[主题 CSS 自定义](/theme-custom-css)**：站方前台自定义 CSS（覆盖顺序、Card 契约、示例）
- **[通知设置](/notification-settings)**：告警通知配置
- **[日志管理](/log-management)**：查看/轮转/清理日志文件
- **[节点分享访问](/stats-share-tokens)**：创建/撤销节点详情临时分享链接并查看访问日志
- **[SSH脚本](/ssh-scripts)**：批量执行脚本管理
- **[高级设置](/advanced-settings)**：高级功能配置
- **[许可证管理](/license-management)**：许可证激活和功能升级

#### 安全功能
- **[安全功能总览](/security-features)**：查看面板当前支持的安全能力和建议开启顺序
- **[安全设置](/security-settings)**：2FA、GitHub 登录、动态入口、HTTP 安全、CSP 等配置
- **[登录与入口](/login-management)**：登录页、动态入口和管理员登录说明
- **[本地 SSH 保险箱](/ssh-local-vault)**：把 SSH 认证保存在当前浏览器
- **[前后台域名分离](/domain-split-manual)**：前台展示域名与后台管理域名分开部署
