# 分组管理页面

## 📍 页面路径
管理后台 → 分组管理（`/admin/groups`）

## 🎯 页面功能
- 创建和管理服务器分组
- 设置默认分组
- 拖拽调整分组顺序
- 删除分组（组内服务器自动移动到默认分组）

## 📋 页面布局

### 分组列表
显示所有分组信息：
- **分组名称**：分组的显示名称
- **服务器数量**：该分组包含的服务器数量
- **默认标记**：显示是否为默认分组（默认分组有"默认"标签和星标图标）
- **操作按钮**：编辑、设为默认、删除（默认分组只能编辑，不能删除）

### 分组操作
- **新增分组**：创建新的分组
- **编辑分组**：修改分组名称
- **设为默认**：将分组设置为默认分组（新服务器默认加入此分组）
- **删除分组**：删除分组（组内服务器自动移动到默认分组）
- **拖拽排序**：通过拖拽分组左侧手柄调整显示顺序

## 🔧 操作步骤

### 创建新分组
1. 点击页面右上角的"新增分组"按钮
2. 在弹出的对话框中输入分组名称
3. 点击"保存"按钮
4. 分组创建成功，页面自动刷新显示新分组

### 编辑分组名称
1. 在分组列表中，将鼠标悬停在要编辑的分组上
2. 点击分组右侧的"编辑"按钮（铅笔图标）
3. 在弹出的对话框中修改分组名称
4. 点击"保存"按钮
5. 修改成功，页面自动刷新

### 设置默认分组
1. 在分组列表中，将鼠标悬停在要设为默认的分组上（非默认分组）
2. 点击分组右侧的"设为默认"按钮（星标图标）
3. 确认设置操作
4. 设置成功，该分组显示"默认"标签，之前的默认分组取消默认状态

### 删除分组
1. 在分组列表中，将鼠标悬停在要删除的分组上（非默认分组）
2. 点击分组右侧的"删除"按钮（垃圾桶图标）
3. 确认删除操作（系统会提示组内服务器将被移动到默认分组）
4. 删除成功，该分组被移除，组内服务器自动移动到默认分组

### 调整分组顺序
1. 将鼠标悬停在分组左侧的拖拽手柄上（显示垂直点图标）
2. 按住鼠标左键并拖动分组到目标位置
3. 释放鼠标，分组移动到新位置
4. 排序自动保存，页面显示新的顺序

## ⚠️ 注意事项

- **默认分组**：系统必须有一个默认分组，默认分组不能被删除
- **删除分组**：删除分组时，组内所有服务器会自动移动到默认分组
- **分组排序**：通过拖拽调整顺序后会自动保存，分组列表按排序值降序显示（默认分组始终优先显示）
- **分组用途**：分组用于对服务器进行分类管理，可在服务器管理页面将服务器分配到不同分组

## 💡 使用建议

- **创建分组**：根据服务器用途或位置创建分组（如：生产环境、测试环境、国内服务器、海外服务器等）
- **默认分组**：建议将最常用的分组设置为默认分组，方便新服务器自动加入
- **分组命名**：使用清晰的分组名称，便于识别和管理
- **分组排序**：将重要的分组拖拽到列表上方，方便快速访问
