# 快速开始

## 系统要求

- **操作系统**：Linux
- **权限**：需要 root（或可使用 sudo 的账号）执行安装脚本
- **网络**：需要能访问 `down.vps.mom`（脚本会从这里拉取安装资源）
- **端口**：默认使用 `5555`（如你自定义了端口，以你的端口为准）

安装脚本默认使用 Docker 部署，并会在缺少 Docker 时尝试自动安装。

## 安装步骤

1. 登录并部署：打开 `https://client.vps.mom`，点击“部署”，复制脚本到你的服务器执行

![部署](/deployment.png)

提示：部署脚本会从 `down.vps.mom` 拉取安装资源，请确保你的服务器能访问该域名。

2. 访问系统：打开浏览器访问 `http://<面板主机名或域名>:5555`（如果你自定义了端口，改成你的端口）。

## 常见问题

### 访问不了安装资源

现象：脚本提示下载失败。

处理：先确认服务器能访问 `down.vps.mom`，再重新执行脚本。

### 装完访问不了面板

现象：浏览器打不开 `:5555`。

处理：
1. 确认你使用的端口（默认 5555）在服务器安全组/防火墙里已放行
2. 确认端口没有被其他程序占用（换一个端口再试）
