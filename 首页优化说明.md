## 🎨 自定义建议

### 修改品牌色
编辑 `.vitepress/theme/custom.css`：

```css
:root {
  --vp-c-brand-1: #your-color;  /* 修改为你的品牌色 */
}
```

### 添加更多 Features
编辑 `index.md` 的 `features` 部分：

```yaml
features:
  - icon: 🎯
    title: 你的标题
    details: 功能描述
    link: /your-link  # 可选
```

### 修改 Hero 图片
```yaml
hero:
  image:
    src: /your-image.png
    alt: 描述文字
```
