# 系统设置（管理后台）

## 入口
管理后台 → 管理设置：`/admin/setting`

## 这里能解决什么
- 改面板 Web 端口（默认 5555）
- 改管理员密码
- 改站点名称/网址/Logo/品牌色
- 设默认视图（全局/桌面/移动：卡片或列表）
- 安装 Agent 下载源（安装脚本下载失败时用；页面里仍显示为 neko-status）
- 开关调试模式（日志会更详细）

## 最短操作

### 1）改端口（改完需要重启）
1. 修改“监听端口”，点击“保存设置”
2. 重启面板服务后，再用新端口访问

### 2）安装 Agent 下载失败（换源再重试）
1. 在“客户端下载地址配置”选择代理，或把“下载地址”换成你自己的镜像前缀
2. 回到服务器管理页，重新执行“初始化/安装 Agent”

## 常见问题

### 保存了端口，但访问还是旧端口
因为面板启动时只读取一次端口，所以改端口后必须重启面板服务。

### 忘记管理员密码
在 DStatus 安装目录执行下面命令（脚本会重置管理员密码，并在屏幕上提示新的密码）：

```bash
curl -fsSL https://down.vps.mom/scripts/reset-password.sh | bash
```

登录后，回到本页把密码改成你自己的。
