# 本地 SSH 保险箱

## 这是什么

本地 SSH 保险箱会把密码或私钥只保存在当前浏览器里。

这样你既不用反复重填，也不用把这些内容长期留在面板里。

## 支持哪些功能

- 测试连接
- SSH 安装更新
- 终端连接
- 终端里直接执行命令
- 终端里的 Agent

## 怎么用

- 你已经手动填了密码或私钥
  点 `保存到本地`
- 面板里已经保存了密码或私钥
  点 `迁移到本地`
- 需要备份当前浏览器里的本地数据
  点 `导出`
- 需要恢复本地数据
  点 `导入`

## 数据怎么走

```text
                    你发起操作
                         |
                         v
              当前页面有没有手动输入
                  /                \
                 /有                \没有
                v                    v
            直接使用          这个浏览器里有没有本地认证
                                  /                \
                                 /有并且已解锁      \没有
                                v                    v
                        优先使用本地认证      面板里有没有已保存认证
                                                  /              \
                                                 /有              \没有
                                                v                  v
                                            直接使用          再让你临时输入一次
```

## 终端怎么用

如果你已经在服务器管理页保存过本地认证，终端页会自动优先使用这一份本地认证。

通常不用再重填一次。

如果你在终端里点了 `清除临时配置`，只会清掉这次终端连接用到的临时信息，不会清掉浏览器里的本地 SSH 保险箱。

## 注意事项

- 只属于当前浏览器
- 换浏览器或换电脑后不会自动带过去
- 需要先导出，再到新环境导入
- 保险箱口令忘记后无法找回
- 导出时会导出这个浏览器里的全部本地数据
- 导入时会替换这个浏览器里原有的本地数据
