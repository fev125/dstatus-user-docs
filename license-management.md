# 许可证管理页面

## 📍 页面路径
管理后台 → 许可证管理（`/admin/license-management`）

## 🎯 页面功能
- 查看当前许可证状态和到期时间
- 激活新的许可证密钥
- 刷新许可证状态
- 解绑当前许可证
- 跳转到升级/购买页面

## 📋 页面布局

### 许可证状态显示区域
显示当前许可证的详细信息：
- **套餐名称**：当前许可证套餐类型
- **到期时间**：许可证到期日期（永久许可证显示"永久"）
- **节点限制**：当前使用的节点数 / 最大允许节点数
- **实例ID**：当前服务器实例的唯一标识符
- **许可证密钥**：当前激活的许可证密钥
- **功能列表**：当前套餐包含的功能模块
- **连接状态**：与许可证服务器的连接状态（在线/缓存/限制/断开）

### 许可证激活区域
未激活许可证时显示：
- **许可证密钥输入框**：输入许可证密钥
- **激活按钮**：点击激活许可证

### 许可证管理操作
已激活许可证时显示：
- **刷新状态**：强制刷新许可证状态（在线验证）
- **解绑许可证**：从当前实例解绑许可证

### 升级/购买
- **升级许可证**按钮：跳转到升级/购买页面

## 🔧 操作步骤

### 查看许可证状态
1. 进入"许可证管理"页面
2. 页面自动加载并显示当前许可证状态
3. 查看连接状态指示器：
   - 🟢 **在线**：与许可证服务器连接正常
   - 🟡 **缓存**：使用本地缓存数据，正在尝试重新连接
   - 🔴 **限制**：使用应急缓存，功能受限
   - ⚪ **断开**：无法连接许可证服务器

### 激活许可证
1. 在"许可证激活"区域输入许可证密钥
2. 点击"激活"按钮
3. 等待验证结果（通常需要几秒钟）
4. 激活成功后页面自动刷新显示许可证信息

### 刷新许可证状态
1. 点击"刷新状态"按钮
2. 系统强制在线验证许可证状态
3. 更新本地缓存和显示信息

### 解绑许可证
1. 点击"解绑许可证"按钮
2. 确认解绑操作
3. 解绑后当前实例将不再使用该许可证

### 升级/购买许可证
1. 点击"升级许可证"按钮
2. 页面跳转到升级/购买页面
3. 选择套餐并完成购买

## 🚨 常见问题

### 许可证激活失败
**可能原因**：
- 许可证密钥无效或已过期
- 网络连接问题
- 许可证服务器不可访问

**解决方法**：
1. 检查许可证密钥格式是否正确
2. 检查网络连接和防火墙设置
3. 查看页面错误提示信息
4. 尝试刷新页面后重新激活

### 连接状态显示异常
**可能原因**：
- 许可证服务器网络中断
- 防火墙阻止连接
- 服务器时间不正确

**解决方法**：
1. 检查服务器网络连接
2. 检查防火墙规则（确保可访问许可证服务器）
3. 检查服务器系统时间是否正确
4. 使用"刷新状态"功能强制重新连接

### 功能列表显示不完整
**可能原因**：
- 使用缓存数据
- 连接状态为"限制"或"断开"

**解决方法**：
1. 点击"刷新状态"强制在线验证
2. 检查连接状态是否恢复正常
3. 如果持续异常，检查网络和服务器配置
